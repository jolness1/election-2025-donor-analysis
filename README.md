# 2025 Mayoral Donor Analysis

### Purpose:
- Determine what the donors for each campaign normally gave to

### Races analyzed:
- Billings Mayoral — { Jennifer Owen v. Mike Nelson (winner) }
- Kalispell Mayoral — { Kisa Davison v. Ryan Hunter (winner) v. Sid Daoud }

### Uses:
- C7 Forms from candidates (aggregated in to .csv files)
- [FollowTheMoney](followthemoney.org) for donor histories
